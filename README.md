# Outgoing Call Reports

A lightweight WordPress plugin for registering, managing, and tracking outgoing call reports directly from the WordPress dashboard.

This plugin creates a centralized reporting system for sales, support, customer success, and HR teams, allowing users to submit call reports through a dashboard widget while managers can review all reports from a dedicated admin section.

---

## Features

- Custom Post Type for outgoing call reports
- Dashboard widget for quick report submission
- AJAX-powered report creation
- Custom call categories
- Custom call status management
- Role-based permissions
- Dedicated admin panel
- Custom archive columns
- Sortable columns
- Automatic reporter tracking
- Read-only reports after submission
- Internal reporting workflow
- Persian-friendly interface
- Lightweight and easy to customize

---

## Call Categories

- Product Back In Stock
- Abandoned Cart Recovery
- Product & Service Suggestions
- HR & Recruitment
- Pre-Purchase Customer Follow-Up
- Registered Users Without Orders
- Old Customer Happy Call
- VIP Customer Happy Call
- Subscription Renewal Follow-Up
- Other

---

## Call Statuses

- Call Not Connected
- Line Busy
- Call Successfully Connected

---

## Development Notes

This plugin was designed as a lightweight internal call reporting solution for WordPress websites and WooCommerce stores.

The architecture intentionally avoids external CRM dependencies and relies entirely on native WordPress functionality.

### Technical Architecture

- Custom Post Type based storage
- AJAX form submission
- Capability-based access control
- Dashboard Widget integration
- Custom admin columns
- WordPress Roles & Permissions
- Minimal database overhead
- Native WordPress APIs only

### Customization

Developers can easily:

- Add new call categories
- Add custom call statuses
- Create custom reports
- Extend permissions
- Integrate with CRM systems
- Add notifications and automations
- Connect to external APIs

---

## Requirements

- WordPress 6.0+
- PHP 7.4+
- Administrator or Editor role

---

## Installation

1. Upload plugin files to `/wp-content/plugins/`
2. Activate the plugin from WordPress Admin
3. Open the Dashboard
4. Start submitting outgoing call reports

# گزارش تماس‌های خروجی

یک افزونه سبک وردپرسی برای ثبت، مدیریت و پیگیری تماس‌های خروجی از داخل پیشخوان وردپرس.

این افزونه یک سیستم متمرکز برای ثبت گزارش تماس‌های فروش، پشتیبانی، منابع انسانی و ارتباط با مشتری ایجاد می‌کند تا اعضای تیم بتوانند تماس‌های انجام شده را ثبت کرده و مدیران به آرشیو کامل گزارش‌ها دسترسی داشته باشند.

---

## امکانات

- ایجاد نوع پست سفارشی برای گزارش تماس‌ها
- ثبت گزارش از طریق ویجت پیشخوان وردپرس
- ثبت اطلاعات به صورت AJAX
- دسته‌بندی تماس‌ها
- مدیریت وضعیت تماس
- سیستم دسترسی مبتنی بر نقش کاربران
- بخش مدیریت اختصاصی
- ستون‌های سفارشی در لیست گزارش‌ها
- قابلیت مرتب‌سازی ستون‌ها
- ثبت خودکار نام ثبت‌کننده گزارش
- جلوگیری از ویرایش مستقیم گزارش‌ها
- مناسب برای استفاده سازمانی
- رابط کاربری فارسی
- سبک، سریع و کم‌مصرف

---

## انواع تماس

- محصول مجدد موجود شده
- ریکاوری سبد خرید پرداخت نشده
- پیشنهاد محصولات و خدمات
- امور منابع انسانی و استخدام
- خریدار دارای سوال پیش از خرید
- کاربر ثبت‌نام کرده بدون سفارش
- یوزر قدیمی / هپی کال
- یوزر پر خرید / هپی کال
- پیگیری تمدید سفارش
- سایر موارد

---

## وضعیت‌های تماس

- تماس برقرار نشد
- شماره اشغال بود
- تماس با موفقیت برقرار شد

---

## توضیحات توسعه

این افزونه با هدف ایجاد یک سیستم داخلی ثبت تماس توسعه داده شده و هیچ وابستگی به CRM های خارجی ندارد.

تمام اطلاعات با استفاده از ساختارهای استاندارد وردپرس ذخیره و مدیریت می‌شوند.

### معماری فنی

- استفاده از Custom Post Type
- ارسال اطلاعات با AJAX
- سیستم دسترسی مبتنی بر Capability
- ویجت اختصاصی پیشخوان
- ستون‌های سفارشی مدیریت
- استفاده از API های داخلی وردپرس
- حداقل مصرف منابع سرور

### توسعه و شخصی‌سازی

توسعه‌دهندگان می‌توانند به سادگی:

- انواع تماس جدید اضافه کنند
- وضعیت‌های تماس را توسعه دهند
- گزارش‌های مدیریتی ایجاد کنند
- سیستم را به CRM متصل کنند
- اعلان‌های خودکار اضافه کنند
- API های خارجی را متصل کنند
